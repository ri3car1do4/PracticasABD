# PracticasABD
Prácticas  de la asignatura de Ampliación de Bases de Datos (ABD) de la Universidad Complutense de Madrid (UCM).

Tecnologías:
- SQL
- Flask
- PostgreSQL
- Neo4j
- MongoDB

Contenido:
- Ejercicios SQL - PostgreSQL
- Ejercicios ORM
- Ejercicios MongoDB
- Ejercicios Neo4j
- Práctica 1: Implementación de un Sistema de Gestión de Bases de Datos (SGDB)
- Práctica 2: Aplicación de Cartas de NBA (Flask + MongoDB)
- Práctica 3: Aplicación de Quiz de Eurovisión (Flask + Neo4j)
